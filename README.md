# Kavor Industries Website

This is a static corporate website for Kavor Industries.

## Files
- index.html
- styles.css
- script.js
- assets/kavor-symbol.svg
- assets/favicon.svg

## How to preview
Open `index.html` in your browser.

## How to launch
Upload these files to any static hosting provider such as:
- Hostinger
- GoDaddy hosting
- Namecheap hosting
- Netlify
- Vercel
- GitHub Pages

## Notes before launch
- Register the domain: kavorindustries.com
- Create professional email: info@kavorindustries.com
- Replace demo contact form mailto if you want a backend form.
- Add legal pages before launch: Privacy Policy, Terms, Imprint if targeting Germany.
